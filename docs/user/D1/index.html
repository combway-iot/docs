Introduction

Brief Introduction​

D1 is an industrial-grade 4G LTE cellular router with a 5-port Ethernet interface for mission-critical communications.

D1 is featured with multiple network protocols and VPN tunnels, provides secure and reliable solutions for a range of M2M/IoT deployments.

D1 is built with hardened enclosure, industrial components, and wide operation range from -40°C to 70°C for harsh environments. Aside from Dual SIM redundancy and Wi-Fi for seamless wireless connectivity, it is also equipped with 4 PSE PoE ports, serial ports, digital I/O for field communications.
